The tool does not require extra dependencies beside standard Odoo apps

When you send a message for your customer or for your colleague, you do not obligatory want such partners to follow a current discussion. It might be both dangerous and annoying. This tool solves the problem. The app stops automatic subscription for recipients, but still let users subscribe them manually in a single click.

Full control over followers. No accidental spam

The tool changes the default Odoo behaviour: message recipients are not any more subscribed for documents. At least if you do not purposefully make them following. To that end it is always possible to check the box 'Subscribe recipients' while composing an email.
With the app users also might send messages without a need to be a follower or a need to unsubscribe each time. The feature would be highly appreciated by managers which control many inferiors.

All cases to be confident

Auto subscription is stopped for both manual messages and for business emails. For example, the buttons 'Send quotation' (sale order), 'Send invoice' (account invoice), 'Send RFQ' (purchase order) and others would not any more result in auto following by recipients or by an author.
The app works independent on which set of Odoo modules you have installed, and doesn't stop auto subscription which is required for business logic.

Which Odoo work flows should not be cancelled

Some Odoo business processes relies upon auto subscription. Those work flows should have auto subscription to guarantee correct security rights and valid users involvement.

That's why the tool doesn't influence business logic, but re-defines behaviour of messages composed by users. In particular, when Odoo fetches an email to create a support ticket, its author should be subscribed. Otherwise, helpdesk might be broken. Another example is the case when you add a customer to a project. Such a partner should be a follower or this contact would not have enough security rights.

How should I install your app?

  1. Unzip source code of purchased tools in one of your Odoo add-ons directory

  2. Re-start the Odoo server

  3. Turn on the developer mode (technical settings)

  4. Update the apps' list (the apps' menu)

  5. Find the app and push the button 'Install'

  6. Follow the guidelines on the app's page if those exist.
