Hide product description in Sale, Purchase and Accounting

This module hides description in:
-Sales Orders/Quotations
-Purchase Order Lines
-Invoice Lines

Technical information/Known Issues:
-Sales Orders/Quotations - module actually swaps "Product" with "Description" in order to eliminate error with not displaying the "Note" or "Section"
-Purchase Order Lines - Section/Note may be not displayed in the list, this does not affect the pdf documents
-Invoice Lines- Section/Note may be not displayed in the list, this does not affect the pdf documents
